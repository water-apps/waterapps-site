A transparent, line-by-line breakdown of what a production-ready, APRA-grade AWS infrastructure footprint actually costs — and what you get for it.

Why Cost Transparency Matters in Procurement

One of the most common friction points in cloud adoption conversations with regulated enterprises isn't security, compliance, or architecture — it's cost uncertainty. "What will this actually cost us?" is the question that stalls projects.

Most cloud vendors quote aspirational figures. Most consultants are vague. This article names specific numbers for a specific stack, built to enterprise-grade standards, that WaterApps uses as a reference architecture for client engagements.

The range is $100–400 USD/month depending on traffic, data volume, and environment size. Here's what you get — and why each component is in the stack.

Costs are estimates based on ap-southeast-2 (Sydney) pricing as of 2026. Actual costs vary with usage patterns. All figures in USD.

The Stack: 7 Modules

1. Networking (VPC)

~$15–30/mo

3-tier VPC with public, private, and isolated subnets across 3 Availability Zones. NAT Gateways for outbound traffic from private subnets. VPC Flow Logs to S3 for network audit trail.

  • NAT Gateway: ~$10–20/mo (2 AZs for HA) — largest networking cost
  • VPC Flow Logs storage: ~$2–5/mo depending on traffic volume
  • VPC endpoints for S3, ECR, SSM: free after transfer savings

2. Compute (ECS Fargate)

~$30–80/mo

ECS Fargate for containerized workloads. No EC2 instances to patch, no bastion hosts to manage. Application Load Balancer in the public subnet, tasks in private subnets.

  • Fargate: 0.25 vCPU / 0.5 GB RAM running 24/7 ≈ $10–12/mo per task
  • Application Load Balancer: ~$18/mo base + LCU usage
  • ECR private registry: ~$0.10/GB/mo for image storage

3. Database (RDS PostgreSQL)

~$50–120/mo

RDS PostgreSQL in Multi-AZ for production resilience. db.t3.medium is the smallest instance we deploy to production — anything smaller creates operational risk in regulated environments. Encrypted at rest with a customer-managed KMS key.

  • db.t3.medium Multi-AZ: ~$80–100/mo (the biggest cost driver)
  • Storage: 20 GB gp3 ≈ $3/mo
  • Automated backups: included in storage costs
  • Single-AZ for dev/staging saves ~50% but is not acceptable for production

4. CDN & Edge (CloudFront)

~$5–20/mo

CloudFront distribution in front of the ALB for TLS termination, DDoS protection via AWS Shield Standard, and edge caching of static assets. WAF with managed rule groups for OWASP Top 10 protection.

  • CloudFront: ~$1–10/mo for low-moderate traffic (first 10TB free tier applies)
  • WAF: ~$5/mo base + $1/mo per rule group
  • ACM certificate: free when used with CloudFront

5. Key Management (KMS)

~$5–10/mo

Customer-managed KMS keys for RDS, S3 (Terraform state, logs, backups), ECS secrets. CMKs are non-negotiable for regulated workloads — they give you key ownership and enable key policy controls that AWS-managed keys don't.

  • $1/mo per CMK + $0.03 per 10,000 API calls
  • Typically 3–5 keys in a standard stack: RDS, S3, application secrets
  • Key rotation: automated, no additional cost

6. Monitoring & Observability

~$10–30/mo

CloudWatch Log Groups for application and infrastructure logs, with 90-day retention (APRA minimum). CloudWatch Alarms on RDS CPU, ECS memory, ALB 5xx errors, and KMS usage. CloudTrail enabled across all regions.

  • CloudWatch Logs: ~$0.57/GB ingested, $0.03/GB stored
  • CloudWatch Alarms: $0.10/alarm/mo — typically 10–20 alarms in a base stack
  • CloudTrail: first trail free, S3 storage ~$2–5/mo
  • SNS for alerting: negligible at low volume

7. CI/CD Pipeline

~$0–15/mo

GitHub Actions with OIDC federation to AWS. No CodePipeline, no CodeBuild unless client mandates it — GitHub Actions is cheaper and more flexible for most regulated workloads. Workflows for plan, apply, and security scanning.

  • GitHub Actions: included in GitHub Team/Enterprise; ~2,000 free minutes/mo on free plans
  • No CodePipeline or CodeBuild costs
  • S3 for Terraform plan artifacts: negligible

What This Stack Does Not Include

Knowing the ceiling matters too. These are common additions that push costs above the base range:

  • ElastiCache / Redis: Add $30–80/mo for session caching or queue offload.
  • Additional Fargate tasks: Every production service adds $10–20/mo per task running 24/7.
  • Data transfer: Significant cross-AZ or internet egress traffic adds up quickly. NAT Gateway egress is $0.059/GB — model your traffic patterns.
  • Secrets Manager: $0.40/secret/mo + $0.05 per 10k API calls. For most apps, $5–15/mo additional.
  • AWS Config: $0.003 per configuration item — typically $5–20/mo for a stack this size. Valuable for compliance evidence.
  • Security Hub: $0.0010 per finding record/mo. Essential for APRA workloads but adds $10–40/mo depending on rule set.
